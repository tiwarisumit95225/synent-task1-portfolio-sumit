# Sumit Tiwari Portfolio

A responsive personal portfolio website built using HTML, CSS, and JavaScript to showcase my skills, projects, achievements, and contact information.

## Features

* Responsive Design
* Modern Dark UI
* Interactive Project Image Modal
* About Me Section
* Skills Showcase
* Project Portfolio
* Contact Section
* Resume Download
* Git Version Control

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Git
* GitHub
